# 📊 개인 성적 관리 시스템

## 📌 프로젝트 개요
학생의 성적을 효율적으로 관리하고 분석할 수 있도록 돕는 프로그램입니다.  
성적 입력, 수정, 평균 계산 등의 기능을 제공합니다.

## 🚀 주요 기능
- 학생 정보 관리
- 성적 입력 및 수정
- 평균 및 성적 계산
- 파일 기반 데이터 저장

## 🛠 개발 환경
- 개발 언어: C
- 데이터 저장 방식: 파일 시스템 (TXT)
